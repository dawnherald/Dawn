# 系统分区

- 格式化为了写入文件

- windows nfts linux ext4

- windows：block分区 格式块 等大4kb可改，ID号，分配盘符（windows中概念）

- linux：block分区格式块 等大4kb可改，ID号起设备文件名，分配盘符（挂载点，用目录作为挂载点）

- /--表示根目录   /dev/hda1(表示IDE硬盘接口a第一个硬盘)

- 逻辑分区只能从5开始 ‘

- 必须分区：根分区 swap分区

- /boot 单独分区


# 设备文件名

- /dev/hda1    dev保存硬件设备  表示IDE硬盘接口 a代表第一个硬盘，1代表第一个分区
- 挂载（盘符和分区连接在一起）：必须分区
  1. /根分区
  2. swap分区（2GB）
- 推荐分区
  1. /boot（，启动分区200mb）
  2. 逻辑上都在根目录下，但是实际上在分区里面的内存

# linux命令格式

- ~代表家 每个目录都有一个家  pwd显示位置也就是家目录
- ‘#’ 说明是超级用户，$是普通
- ls 显示 ls -l查看详细信息
- rw- u所有者  r--g所属组 r--o其他人      为了设置权限
- r 读 w 写 x 执行 
- 硬链接 一个文件的两个门
- locate速度很快  只能搜索文件名 
- 搜索命令的命令 whereis 搜索命令所在文件地址 看到帮助文档
- which 查看别名 以及地址
- echo $PATH 系统搜索命令的路径
- find  最强大 
  1. 避免过大范围搜索
  2. 搜索内容和name一摸一样 除非使用通配符
  3. find 查找        kb 要小写 M要大写
- 目录创建 mkdir 创建文件 touch
